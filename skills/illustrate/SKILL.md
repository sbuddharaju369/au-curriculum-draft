# illustrate

**When:** a concept is clearer with a runnable demo (e.g. measuring memory bandwidth,
tokenizing a prompt).

**Procedure:** write a small, focused script to `scratch/python/` or `scratch/pytorch/`,
run it, and explain the output in plain language tied back to the lesson.

**Output contract:** the script is committed under `scratch/` (exploration is signal and
portfolio). No edits outside `scratch/`.

> Driven from the agent's system prompt + `agents.md`.
